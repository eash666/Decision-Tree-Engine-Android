# 🌳 Decision Tree Inference Engine (Android)

> An out-of-the-box Android application that dynamically parses text-based rule configurations into visual, interactive decision trees.

## 🚀 Overview
This project transforms a traditional command-line decision tree system into a modern, interactive mobile application. It is designed to demonstrate strong foundational skills in **Data Structures, Algorithm Design (Tree Traversal/State Machines), and End-to-End Software Engineering**.

## ✨ Core Features
- **Dynamic Parsing:** Reads arbitrary `.txt` configuration files and dynamically builds a multi-way tree structure in memory.
- **State Machine Interaction:** Supports node-by-node traversal, path backtracking, and full preorder traversal output.
- **Secure Data Persistence:** Utilizes Android's Storage Access Framework (SAF) and I/O streams to securely copy and isolate user-imported rules into the app's internal sandbox.
- **Zero-Config Deployment:** Automatically extracts and deploys default decision trees (e.g., Washing Machine Guide) from `assets` upon first launch for an out-of-the-box experience.
- **Modern UI Architecture:** Features a clean, iOS-inspired card layout with dynamically injected UI components based on tree depth.

## 🛠 Tech Stack
- **Language:** Java
- **Framework:** Android SDK
- **Architecture:** Custom Tree Data Structures, State Machine, XML Layouts
- **Tools:** Android Studio, Gradle
